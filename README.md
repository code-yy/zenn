## コマンドメモ

### 以下のコマンドにより markdown ファイルを簡単に作成できます。

`$ npx zenn new:article`

### 本文の執筆は、ブラウザでプレビューしながら確認できます。ブラウザでプレビューするためには次のコマンドを実行します。

`$ npx zenn preview`
